# Aswin Manoj - Personal Portfolio & Resume

A premium, interactive 3D portfolio website showcasing my qualifications, certifications, projects, and skills. Built with modern frontend technologies including React, Vite, Three.js, and Framer Motion.

🚀 **[Live Demo](https://my-porfolio-seven-iota.vercel.app)**

---

## 🛠️ Tech Stack & Libraries

- **Frontend Core:** React 19, JavaScript (ES6+), HTML5, Vanilla CSS3
- **3D Graphics & Animations:** Three.js, `@react-three/fiber`, `@react-three/drei`
- **Transitions & Interactive Animations:** Framer Motion
- **Scroll Optimization:** Lenis (Smooth Scroll)
- **Icons:** Lucide React
- **Build Tool:** Vite

---

## 🌟 Key Features

- **Interactive 3D Canvas Background:** Dynamic particles and interactive 3D elements powered by Three.js.
- **Glassmorphism Design:** A modern dark-themed user interface utilizing sleek glassmorphic components, custom gradients, and clean borders.
- **Smooth Scroll Experience:** Seamless animations and page transitions powered by Lenis smooth scrolling.
- **Interactive Sections:**
  - **About:** Introduction and professional summary.
  - **Education:** Timeline of my academic progress (IMCA at DiST, MG University).
  - **Internships:** Work history and development experiences.
  - **Projects:** Interactive cards with links to source code and technologies.
  - **Skills:** Categorized skills with dynamic visual cards.
  - **Certifications:** Verified achievements from HackerRank, Coursera, Anthropic, and more.
  - **Contact:** Sleek, integrated contact form and social profiles.

## 💻 Local Development Setup

To run this portfolio locally on your machine, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/aswinmanoj-2004/my-porfolio.git
   cd my-porfolio
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the local development server:**
   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```
